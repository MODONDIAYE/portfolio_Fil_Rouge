UI Application Fitness

Description

Conception d'une interface utilisateur moderne et interactive pour une application de fitness. Le projet inclut la création de maquettes et prototypes pour le suivi des entraînements, des objectifs santé et nutrition, ainsi qu’un tableau de bord personnalisé pour les utilisateurs. L’interface est conçue pour être intuitive et responsive, offrant une expérience fluide sur mobile et tablette.

Technologies utilisées

Figma Adobe XD React Tailwind CSS Illustrator Photoshop
