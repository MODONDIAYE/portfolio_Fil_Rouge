E-commerce Sport – Vente d’équipements sportifs

Ajouté en 2026

Description

Développement d'une plateforme e-commerce spécialisée dans la vente d’équipements sportifs (chaussures, vêtements, accessoires de fitness, matériel de sport). L’application permet aux utilisateurs de parcourir les produits, filtrer par catégories, ajouter des articles au panier et effectuer des paiements en ligne sécurisés. Un espace administrateur permet de gérer les produits, suivre les commandes et analyser les performances de vente. Le site est entièrement responsive et optimisé pour offrir une expérience utilisateur fluide sur mobile et desktop.

Technologies utilisées

React.js Node.js Express.js MongoDB Stripe API JWT Auth Docker AWS / Heroku

Fonctionnalités clés

  • Catalogue produits avec filtres (prix, catégorie, marque)
  • Panier dynamique et gestion des commandes
  • Paiement en ligne sécurisé
  • Interface admin (CRUD produits)
  • Responsive design mobile-first
