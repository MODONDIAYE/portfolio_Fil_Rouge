CashPay – Application Mobile Money

Description

CashPay est une application mobile de paiement et transfert d'argent, permettant aux utilisateurs d'envoyer et recevoir de l'argent instantanément, de payer des factures, et de gérer leur portefeuille en toute sécurité. L'application inclut des fonctionnalités comme le suivi des transactions, notifications en temps réel, codes QR pour les paiements et une interface intuitive adaptée à tous les appareils. Elle est conçue pour offrir rapidité, sécurité et fiabilité aux utilisateurs, avec un focus sur la protection des données personnelles.

Technologies utilisées

React Native Node.js / Express MongoDB Stripe / PayPal JWT Auth Firebase Docker AWS / Heroku
