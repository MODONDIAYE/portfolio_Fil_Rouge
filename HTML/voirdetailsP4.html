Plateforme E-commerce Full Stack

Description

Développement d'une plateforme e-commerce complète avec gestion des produits, panier d'achat, paiements en ligne et tableau de bord administrateur. L'application supporte plusieurs devises et langues, intègre un système de recommandation basé sur l'IA et offre une expérience utilisateur fluide sur tous les appareils.

Technologies utilisées

  • React
  • Node.js
  • PostgreSQL
  • Stripe API
  • Docker
  • Redis
