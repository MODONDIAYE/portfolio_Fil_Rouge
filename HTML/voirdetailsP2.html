Projet 2 : Application de gestion de tâches

Description

Conception et développement d'une application web permettant de gérer efficacement les tâches personnelles et professionnelles. L'application inclut la création, modification et suppression de tâches, la gestion des priorités et des échéances, ainsi qu’un système de notifications et de collaboration en équipe. Elle offre une interface intuitive et responsive pour une utilisation fluide sur ordinateur et mobile.

Technologies utilisées

  • React
  • Node.js
  • MongoDB
  • Express.js
  • Docker
  • JWT (authentification)
